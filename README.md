# mnist_bp
mnist data training and testing with back propagation

# requirement
```
sudo apt-get install python3-pyqt5
pip install opencv-python -i https://pypi.tuna.tsinghua.edu.cn/simple
```

# run
py3
```
python train.py
```
or GUI
```
python UI.py
```

# result
Predicted Correct Rate: 90.94% on test data.
